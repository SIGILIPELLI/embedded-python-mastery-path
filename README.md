# Embedded Python Mastery Path

A free, structured, module-wise MicroPython training program — entry level to
master level, ESP32-first (with Raspberry Pi Pico notes where relevant), with
real, runnable MicroPython code in every module and a hands-on project at the
end of each level. Every Level 1 example runs in the free
[Wokwi online simulator](https://wokwi.com/)'s *MicroPython on ESP32* project
type, so no hardware is required to start.

**Live site:** https://sigilipelli.github.io/embedded-python-mastery-path/

## Contents

- **Level 1 · Entry** — MicroPython vs. Python, REPL & workflow, GPIO, PWM & ADC, timers & interrupts, I2C/SPI sensors, files & persistence, WiFi & HTTP, `uasyncio`, capstone (WiFi sensor logger with web dashboard)
- **Level 2 · Intermediate** — MQTT, BLE with `aioble`, deep sleep & battery power, custom sensor drivers, `mpremote`/`mip` tooling, NTP & timekeeping, NeoPixels & displays, ESP-NOW, on-device logging
- **Level 3 · Advanced** — memory & garbage collector internals, performance profiling, native/viper code emitters, C modules, RP2040 PIO, advanced `uasyncio`, inline assembler, frozen modules, threads
- **Level 4 · Master** — production firmware architecture, OTA for fleets, provisioning & secrets, watchdogs & robustness, custom firmware builds, security hardening, fleet management, manufacturing & factory test

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
